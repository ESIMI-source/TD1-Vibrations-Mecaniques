# TD1 - Vibrations Mécaniques

Ce dépôt contient les énoncés, les rappels théoriques, les solutions détaillées et les simulations Python pour le Travaux Dirigés 1 (TD1) sur les vibrations mécaniques. Il est destiné aux étudiants de 2ème Année CI : GSMI de l'École Nationale Supérieure d'Arts et Métiers (ENSAM) Casablanca.

## Objectifs Pédagogiques

L'objectif de ce TD est de permettre aux étudiants de :

*   Comprendre les concepts fondamentaux des vibrations mécaniques.
*   Appliquer les méthodes de résolution pour les systèmes à un degré de liberté (1 ddl).
*   Analyser les phénomènes d'amortissement et de résonance.
*   Utiliser des outils de simulation numérique (Python/Jupyter) pour visualiser et interpréter les résultats.

## Structure du Dépôt

Le dépôt est organisé comme suit :

*   `README.md` : Ce fichier, présentant le projet.
*   `Theorie/` : Contient les rappels de cours essentiels.
*   `Exercices/` : Contient les énoncés des exercices.
*   `Solutions/` : Contient les corrigés détaillés de chaque exercice.
*   `Simulations/` : Contient les scripts Python (Notebooks Jupyter) pour les simulations.

## Comment Utiliser ce Dépôt

1.  **Cloner le dépôt** :
    ```bash
    git clone [URL_DU_DEPOT]
    ```
2.  **Consulter les énoncés** : Rendez-vous dans le dossier `Exercices/`.
3.  **Étudier la théorie** : Le dossier `Theorie/` contient les rappels nécessaires.
4.  **Travailler les exercices** : Tentez de résoudre les exercices par vous-même avant de consulter les solutions.
5.  **Vérifier les solutions** : Le dossier `Solutions/` propose des corrigés détaillés.
6.  **Exécuter les simulations** : Ouvrez les notebooks Jupyter dans le dossier `Simulations/` pour visualiser les phénomènes.
